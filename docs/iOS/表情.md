# 表情

1. 使用接口 请求所有的表情
2. 表情里面有code note url动态图 static_url静态图

所有的表情图片文件是放到了服务器上的，不是在本地的。

                        note = "[爆笑]";

                    },
                                        {
                        code = "\\\\00125";
