暂存
向回计算 

大量递归占内存（暂存）。

优点：
容易理解，代码简洁。

例：

计算阶乘

> 5! = 5 * 4 * 3 * 2 * 1;
>
> n! = n * (n - 1)!;
>
> 5! = 5 * 4!; //计算5的阶乘，计算不出来，先暂存，去计算4的阶乘。
>
> 4! = 4 * 3!;//同理 先暂存
>
> 3! = 3 * 2!;
>
> 2! = 2 * 1!;
>
> 1! = 1 * 0!;
>
> 0! = 1; //到这里可以计算出来，然后再往回推。把上面暂存的计算。

代码：

```c++
//n! = n * (n-1)!;
long 阶乘(int n)
{
    if (n == 0) {
        return 1;
    }
    else {
        return n * 阶乘(n - 1);
    }
}
```

调用

```c++
void test()
{
    
    std::cout << 阶乘(5) << std::endl;
    for (int num = 0; num < 10; num++) {
        std::cout << num << "!=" << 阶乘(num) << std::endl;
    }
}
```

使用迭代（循环）

```c++
long 阶乘2(int n)
{
    long 结果 = 1;
    for (int i = n; i > 0; i--) {
        结果 = 结果 * i;
    }
    return 结果;
}
```
