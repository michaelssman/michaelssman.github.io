## 快捷键

- ctrl+d：复制一行
- ctrl+y：删除一行
- sout：System.*out*.println();
- alt+Insert：类构造器，可以选择多个属性

## 方法重载

同一个类中，**方法名相同**，**形参列表不同**（类型不同、个数不同、顺序不同）的多个方法，构成了方法的重载。
