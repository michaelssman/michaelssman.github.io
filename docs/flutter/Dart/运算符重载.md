# 运算符重载

```dart
operatorDemo() {
  OperatorClass op1 = OperatorClass(18);
  OperatorClass op2 = OperatorClass(32);
  print(op1 > op2);
}

class OperatorClass {
  int age;
  OperatorClass(this.age);
  bool operator >(OperatorClass other) => this.age > other.age;
}
```
