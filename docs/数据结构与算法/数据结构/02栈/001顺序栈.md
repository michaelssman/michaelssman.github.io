# 栈-顺序栈

堆栈就是一种容器  存储各种数据 

栈是一种数据结构，系统软件或算法中用到栈。

有两种栈：

1. 顺序栈   使用数组实现
2. 链式栈	使用链表实现

- 后进先出（LIFO）或 先进后出（FILO）
- 栈底，栈顶
- 栈的操作
  1. Push（添加到数组中 先判断是否满了  数组满了 要扩大容量 旧数据拷贝到新到里面）
  2. Top
  3. Pop 删除栈顶数据，栈顶指向下一个
  4. IsEmpty

用动态数组设计栈。设计一个工具类实现数组满了 把原来的数组放大一倍。
