# 希尔排序

> 排序过程：
>
> 先取一个正整数d1<n，把所有相隔d1的记录放一组，组内进行直接插入排序；然后取d2<d1，重复上述分组和排序操作；直至di=1，即所有记录放进一个组中排序为至
>
> O(n-1.3)
>
> O(nlogn)
>
> 希尔排序是不稳定的
