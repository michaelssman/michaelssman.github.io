# ai记账使用说明

## 1、账户列表

### 1.1、新增账户

使用应用需要先新增一个账户，新增页面可以设置账户的名称、图标、类别、初始余额。

图标：选择一个图标，可以更加快速的找到账户。

类别：有现金、银行卡、虚拟账户、投资理财、信用卡等，对账户进行分类管理和查看。

### 1.2、账户列表左滑

- 可以删除：删除会将该账户下所有的明细都删除，并且无法撤回。
- 可以编辑：编辑该账户的名称、图标、类别。
