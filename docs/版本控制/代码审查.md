# AI 代码审核

## 操作步骤

### 1、通过git命令提取提交记录

前提条件是需要审核代码的分支（git branch） 

**提取提交记录命令**

 ```sh
 git log --oneline --patch > test_commits_detail.txt
 ```

**限制时间范围命令**

 ```sh
 git log --since="2025-09-04" --until="2025-10-16" --oneline --patch > test_commits_detail.txt
 ```

**限制提交者命令**

```sh
git log --since="2025-08-04" --until="2025-11-11" --author="michael" --oneline --patch >/Users/michael/Desktop/scm.txt
```

### 2、上传文件给AI

豆包、DeepSeek、腾讯元宝（3个选一个即可）。

提问方式：根据git 提交记录请生成一份代码审核报告，包含：概要、修改文件、人工审查重点、风险与建议、修复优先级 https://www.doubao.com/thread/wa6b54ef59e82f147 

可以给出一个已有的文件，然后让ai：按照给出的word文件中的格式，重写一下。

注意：每次提交描述尽可能详细，这样在AI 代码审核时结果更全面。
